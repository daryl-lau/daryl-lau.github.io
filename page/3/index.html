InnodbеӯҳеӮЁеј•ж“Һдёӯзҡ„дәӢеҠЎе®Ңе…Ёз¬ҰеҗҲACIDзҡ„зү№жҖ§пјҡ

  2. дёҖиҮҙжҖ§пјҲconsistencyпјүпјҡдёҖиҮҙжҖ§жҳҜжҢҮдәӢеҠЎе°Ҷж•°жҚ®еә“д»ҺдёҖз§ҚзҠ¶жҖҒиҪ¬еҸҳдёәдёӢдёҖз§ҚдёҖиҮҙзҡ„зҠ¶жҖҒпјҢеңЁдәӢеҠЎејҖе§Ӣд№ӢеүҚе’ҢдәӢеҠЎз»“жқҹд»ҘеҗҺпјҢж•°жҚ®еә“зҡ„е®Ңж•ҙжҖ§зәҰжқҹжІЎжңүиў«з ҙеқҸ.
  3. йҡ”зҰ»жҖ§пјҲisolationпјүпјҡдёҖдёӘдәӢеҠЎзҡ„еҪұе“ҚеңЁиҜҘдәӢеҠЎжҸҗдәӨеүҚеҜ№е…¶д»–дәӢеҠЎйғҪдёҚеҸҜи§Ғ
  4. жҢҒд№…жҖ§пјҲdurabilityпјүпјҡдәӢеҠЎдёҖж—ҰжҸҗдәӨпјҢе…¶з»“жһңе°ұжҳҜж°ёд№…жҖ§.
