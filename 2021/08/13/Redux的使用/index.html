Reduxзҡ„дҪҝз”Ё

еҸ‘иЎЁдәҺ 2021-08-13 еҲҶзұ»дәҺ React жң¬ж–Үеӯ—ж•°пјҡ 3.2k

з®Җд»Ӣ

Redux жҳҜдёҖдёӘдҪҝз”ЁеҸ«еҒҡвҖңactionвҖқзҡ„дәӢд»¶жқҘз®ЎзҗҶе’Ңжӣҙж–°еә”з”ЁзҠ¶жҖҒзҡ„жЁЎејҸе’Ңе·Ҙе…·еә“ е®ғд»ҘйӣҶдёӯејҸStoreпјҲcentralized storeпјүзҡ„ж–№ејҸеҜ№ж•ҙдёӘеә”з”ЁдёӯдҪҝз”Ёзҡ„зҠ¶жҖҒиҝӣиЎҢйӣҶдёӯз®ЎзҗҶпјҢе…¶и§„еҲҷзЎ®дҝқзҠ¶жҖҒеҸӘиғҪд»ҘеҸҜйў„жөӢзҡ„ж–№ејҸжӣҙж–°гҖӮ

дёәд»Җд№ҲиҰҒдҪҝз”Ё Reduxпјҹ
Redux её®дҪ з®ЎзҗҶвҖңе…ЁеұҖвҖқзҠ¶жҖҒ - е“Әдәӣеә”з”ЁзЁӢеәҸзҡ„и®ёеӨҡйғЁеҲҶйғҪйңҖиҰҒзҡ„зҠ¶жҖҒгҖӮ

Redux жҸҗдҫӣзҡ„жЁЎејҸе’Ңе·Ҙе…·дҪҝжӮЁжӣҙе®№жҳ“зҗҶи§Јеә”з”ЁзЁӢеәҸдёӯзҡ„зҠ¶жҖҒдҪ•ж—¶гҖҒдҪ•ең°гҖҒдёәд»Җд№Ҳд»ҘеҸҠеҰӮдҪ•жӣҙж–°пјҢд»ҘеҸҠеҪ“иҝҷдәӣжӣҙж”№еҸ‘з”ҹж—¶жӮЁзҡ„еә”з”ЁзЁӢеәҸйҖ»иҫ‘е°ҶеҰӮдҪ•иЎЁзҺ°гҖӮ Redux жҢҮеҜјжӮЁзј–еҶҷеҸҜйў„жөӢе’ҢеҸҜжөӢиҜ•зҡ„д»Јз ҒпјҢиҝҷжңүеҠ©дәҺи®©жӮЁзЎ®дҝЎжӮЁзҡ„еә”з”ЁзЁӢеәҸе°ҶжҢүйў„жңҹе·ҘдҪңгҖӮ

дёҚеғҸVuexпјҢReduxжҳҜдёҖдёӘзӢ¬з«Ӣзҡ„е·Ҙе…·еә“пјҢдёҚд»…еҸҜд»ҘеңЁReactдёӯдҪҝз”ЁпјҢиҝҳеҸҜд»ҘеңЁе…¶д»–жЎҶжһ¶дёӯдҪҝз”ЁпјҢз”ҡиҮіеҸҜд»ҘзӣҙжҺҘз”Ёscriptж Үзӯҫзҡ„еҪўејҸеңЁйЎөйқўдёӯзӣҙжҺҘдҪҝз”Ё

ReduxдҪҝз”Ё

жҲ‘д»¬е°ұйҖҡиҝҮдёҖдёӘз®ҖеҚ•зҡ„и®Ўж•°еҷЁйЎөйқўжқҘдәҶи§ЈReduxиҜҘжҖҺд№ҲдҪҝз”ЁпјҢжҲ‘д»¬е…ҲдёҚеңЁreactйЎ№зӣ®дёӯдҪҝз”ЁпјҢе…ҲзңӢзңӢжңҖеҺҹе§Ӣзҡ„reduxиҰҒжҖҺд№Ҳз”ЁгҖӮеҗҺйқўеҶҚзңӢзңӢжҖҺд№ҲеңЁreactдёӯдҪҝз”ЁгҖӮ

еҹәзЎҖзҡ„htmlйЎөйқўеҰӮдёӢпјҢйңҖиҰҒеј•з”Ёreduxзҡ„umdеҢ…пјҢпјҲдҪҝз”ЁjqueryжқҘж–№дҫҝжҲ‘д»¬ж“ҚдҪңdomпјү

еј•з”ЁдәҶreduxпјҢдјҡеңЁWindowдёҠжҢӮиҪҪдёҖдёӘReduxе…ЁеұҖеҜ№иұЎпјҢжҲ‘д»¬йҰ–е…ҲйңҖиҰҒе®ҡд№үдёҖдёӘreducerпјҢreducer жҳҜдёҖдёӘеҮҪж•°пјҢжҺҘж”¶еҪ“еүҚзҡ„ state е’ҢдёҖдёӘ action еҜ№иұЎпјҢеҝ…иҰҒж—¶еҶіе®ҡеҰӮдҪ•жӣҙж–°зҠ¶жҖҒпјҢе№¶иҝ”еӣһж–°зҠ¶жҖҒгҖӮеҮҪж•°зӯҫеҗҚжҳҜпјҡ(state, action) => newStateгҖӮ дҪ еҸҜд»Ҙе°Ҷ reducer и§ҶдёәдёҖдёӘдәӢд»¶зӣ‘еҗ¬еҷЁпјҢе®ғж №жҚ®жҺҘж”¶еҲ°зҡ„ actionпјҲдәӢд»¶пјүзұ»еһӢеӨ„зҗҶдәӢд»¶гҖӮ

action жҳҜдёҖдёӘе…·жңү type еӯ—ж®өзҡ„жҷ®йҖҡ JavaScript еҜ№иұЎгҖӮдҪ еҸҜд»Ҙе°Ҷ action и§ҶдёәжҸҸиҝ°еә”з”ЁзЁӢеәҸдёӯеҸ‘з”ҹдәҶд»Җд№Ҳзҡ„дәӢд»¶гҖӮtype еӯ—ж®өжҳҜдёҖдёӘеӯ—з¬ҰдёІпјҢз»ҷиҝҷдёӘ action дёҖдёӘжҸҸиҝ°жҖ§зҡ„еҗҚеӯ—пјҢжҜ”еҰӮ"todos/todoAdded"гҖӮжҲ‘д»¬йҖҡеёёжҠҠйӮЈдёӘзұ»еһӢзҡ„еӯ—з¬ҰдёІеҶҷжҲҗвҖңеҹҹ/дәӢд»¶еҗҚз§°вҖқпјҢе…¶дёӯз¬¬дёҖйғЁеҲҶжҳҜиҝҷдёӘ action жүҖеұһзҡ„зү№еҫҒжҲ–зұ»еҲ«пјҢз¬¬дәҢйғЁеҲҶжҳҜеҸ‘з”ҹзҡ„е…·дҪ“дәӢжғ…гҖӮ

action еҜ№иұЎеҸҜд»Ҙжңүе…¶д»–еӯ—ж®өпјҢе…¶дёӯеҢ…еҗ«жңүе…іеҸ‘з”ҹзҡ„дәӢжғ…зҡ„йҷ„еҠ дҝЎжҒҜгҖӮжҢүз…§жғҜдҫӢпјҢжҲ‘д»¬е°ҶиҜҘдҝЎжҒҜж”ҫеңЁеҗҚдёә payload зҡ„еӯ—ж®өдёӯгҖӮ

еңЁе®ҡд№үreducerзҡ„ж—¶еҖҷпјҢжіЁж„Ҹиҝ”еӣһзҡ„зҠ¶жҖҒеҝ…йЎ»жҳҜдёҖдёӘе…Ёж–°зҡ„зҠ¶жҖҒпјҢдёҚиғҪзӣҙжҺҘдҪҝз”Ёе’Ңд№ӢеүҚзӣёеҗҢзҡ„еј•з”ЁпјҢдҫӢеҰӮдҪҝз”Ёobj.aзӣҙжҺҘдҝ®ж”№aзҡ„еҖјпјҢжҲ–иҖ…array.pushдёҖдёӘж–°зҡ„еҖј

жӯӨж—¶жңүдәҶreducerпјҢжҲ‘д»¬е°ұеҸҜд»ҘдҪҝз”Ёе…ЁеұҖзҡ„ReduxеҲӣе»әдёҖдёӘstoreпјҢ

жҲ‘д»¬еҸҜд»ҘзңӢзңӢstoreдёҠжңүе“ӘдәӣAPI

еҸ‘зҺ°дёҠйқўжңүdispatchгҖҒgetStateгҖҒreplaceReducerгҖҒsubscribeиҝҷеӣӣдёӘAPIпјҢдёӢйқўжҲ‘д»¬йғҪдјҡз”ЁеҲ°гҖӮ

store.getState()

йҖҡиҝҮдёҠеӣҫеҸҜд»ҘеҸ‘зҺ°пјҢжҲ‘д»¬зҡ„йЎөйқўcountеӯ—ж®өжҳҜжІЎеҖјзҡ„пјҢйӮЈжҲ‘д»¬иҰҒжҖҺд№ҲиҺ·еҸ–еҲ°storeдёӯзҡ„stateе‘ўпјҹжӯӨж—¶е°ұеҸҜд»ҘдҪҝз”Ёstore.getState()жқҘиҺ·еҸ–storeдёӯзҡ„зҠ¶жҖҒдәҶпјҢжҲ‘д»¬иҺ·еҸ–еҲ°stateз„¶еҗҺжёІжҹ“еҲ°йЎөйқўдёӯ

store.dispatch

жӯӨж—¶йЎөйқўе°ұе·Із»ҸжңүдәҶcountзҡ„еҖјдәҶпјҢйӮЈжҖҺд№Ҳдҝ®ж”№stateе‘ўпјҹжӯӨж—¶жҲ‘д»¬е°ұеҸҜд»ҘдҪҝз”ЁdispatchжқҘжҙҫеҸ‘дёҖдёӘactionпјҢз„¶еҗҺеңЁreducerдёӯж №жҚ®actionзҡ„typeеҖјпјҢжқҘеҲӨж–ӯйңҖиҰҒжҖҺд№ҲеӨ„зҗҶstate

жҲ‘д»¬зӮ№еҮ»жҢүй’®+пјҢйҖҡиҝҮstore dispatch дёҖдёӘaction {type: 'INCREMENT'}пјҢз„¶еҗҺreducerдёӯе°ұдјҡжҢүз…§actionзҡ„typeзұ»еһӢпјҢеҺ»жӣҙж–°stateпјҢзӮ№еҮ»жҢүй’®-д№ҹжҳҜдёҖж ·

жҲ‘д»¬зңӢзңӢж•Ҳжһң

store.subscribe

жӯӨж—¶stateжҳҜзЎ®е®һиў«жӣҙж–°дәҶпјҢдҪҶжҳҜйЎөйқўдёәд»Җд№ҲжІЎжңүж”№еҸҳе‘ўпјҹиҝҷжҳҜеӣ дёәstateзЎ®е®һжҳҜж”№еҸҳдәҶпјҢдҪҶжҳҜreduxе№¶жІЎжңүе‘ҠиҜүжҲ‘stateж”№еҸҳдәҶе•ҠпјҢжүҖд»ҘжҲ‘д»¬йңҖиҰҒзҹҘйҒ“stateеҸҳеҢ–дәҶпјҢеҶҚжү§иЎҢдёҖдәӣеҠЁдҪңгҖӮжӯӨж—¶е°ұеҸҜд»ҘдҪҝз”Ёstoreзҡ„subscribeжқҘи®ўйҳ…дёҖдёӘзӣ‘еҗ¬еҷЁпјҢдёҖж—Ұstateж”№еҸҳдәҶпјҢreduxе°ұеҺ»жү§иЎҢйӮЈдёӘзӣ‘еҗ¬еҷЁпјҢеҒҡзӣёеә”зҡ„жӣҙж”№гҖӮ

subscribeжҺҘж”¶дёҖдёӘеҮҪж•°дҪңдёәеҸӮж•°пјҢжҲ‘д»¬е°ҶжҲ‘д»¬йңҖиҰҒеҒҡзҡ„ж“ҚдҪңдј иҝӣеҺ»пјҢдёҖж—ҰreduxжЈҖжөӢеҲ°stateеҸ‘з”ҹдәҶеҸҳеҢ–пјҢе°ұдјҡеҺ»и°ғз”ЁиҝҷдёӘеҮҪж•°

subscribeиҝҳжҺҘж”¶дёҖдёӘиҝ”еӣһеҖјпјҢз”ЁжқҘеҸ–ж¶Ҳи®ўйҳ…пјҢиҝҷдёӘиҝ”еӣһеҖјжҳҜдёҖдёӘж–№жі•пјҢжҲ‘д»¬зӣҙжҺҘи°ғз”ЁиҝҷдёӘж–№жі•е°ұеҸҜд»ҘеҸ–ж¶Ҳи®ўйҳ…дәҶпјҢеҗҺз»ӯstateзҡ„дҝ®ж”№дёҚдјҡеҺ»и§ҰеҸ‘зӣ‘еҗ¬еҷЁеҮҪж•°пјҢдҪҶжҳҜreduxеҶ…йғЁstateзҡ„зҠ¶жҖҒиҝҳжҳҜдјҡжҢҒз»ӯжӣҙж–°

жңҖдё»иҰҒзҡ„еҮ дёӘAPIжҲ‘д»¬е·Із»Ҹз”ЁеҲ°дәҶпјҢеү©дёӢзҡ„еҮ дёӘapiжҲ‘д»¬еҲҶеҲ«жқҘиҜҙиҜҙ

Redux.combineReducers

з”ұдәҺдёҚе»әи®®еҲӣе»әеӨҡдёӘstoreпјҢеӣ жӯӨеҰӮжһңжңүеӨҡдёӘreducerпјҢйңҖиҰҒеҲ©з”ЁиҝҷдёӘAPIжқҘз»„еҗҲreducerпјҢ

combineReducers иҫ…еҠ©еҮҪж•°зҡ„дҪңз”ЁжҳҜпјҢжҠҠдёҖдёӘз”ұеӨҡдёӘдёҚеҗҢ reducer еҮҪж•°дҪңдёә value зҡ„ objectпјҢеҗҲе№¶жҲҗдёҖдёӘжңҖз»Ҳзҡ„ reducer еҮҪж•°пјҢз„¶еҗҺе°ұеҸҜд»ҘеҜ№иҝҷдёӘ reducer и°ғз”Ё createStore ж–№жі•гҖӮ

еҗҲе№¶еҗҺзҡ„ reducer еҸҜд»Ҙи°ғз”Ёеҗ„дёӘеӯҗ reducerпјҢе№¶жҠҠе®ғд»¬иҝ”еӣһзҡ„з»“жһңеҗҲе№¶жҲҗдёҖдёӘ state еҜ№иұЎгҖӮ з”ұ combineReducers() иҝ”еӣһзҡ„ state еҜ№иұЎпјҢдјҡе°Ҷдј е…Ҙзҡ„жҜҸдёӘ reducer иҝ”еӣһзҡ„ state жҢүе…¶дј йҖ’з»ҷ combineReducers() ж—¶еҜ№еә”зҡ„ key иҝӣиЎҢе‘ҪеҗҚгҖӮ


еҸҜд»ҘзңӢеҲ°жҲ‘д»¬з»„еҗҲдәҶcountReducerе’ҢpriceReducerпјҢжңҖз»ҲдјҡжҢүз…§дј е…ҘcombineReducersеҜ№иұЎзҡ„keyиҝӣиЎҢе‘ҪеҗҚеҢәеҲҶпјҢеӣ жӯӨжҲ‘д»¬д№ӢеүҚзҡ„д»Јз ҒйңҖиҰҒж”№дёә

жҲ‘д»¬еҸ‘зҺ°дёҖдёӘй—®йўҳпјҢдёәд»Җд№ҲжҲ‘жҙҫеҸ‘дәҶдёҖдёӘactionпјҢдёӨдёӘstateйғҪдјҡеҸ‘з”ҹеҸҳеҢ–пјҹе…¶е®һжҳҜеӣ дёәжҲ‘д»¬зҡ„actionйғҪжҳҜINCREMENTпјҢз”ұдәҺдёӨдёӘreducerдёӯзҡ„typeи®ҫзҪ®зҡ„жҳҜдёҖж ·зҡ„пјҢиҝҷж ·зҡ„иҜқе°ұдјҡеҜјиҮҙиҝҷдёӘй—®йўҳпјҢдёҖиҲ¬дёәдәҶи§ЈеҶіиҝҷдёӘй—®йўҳпјҢжҲ‘д»¬йғҪжҳҜжҠҠйӮЈдёӘзұ»еһӢзҡ„еӯ—з¬ҰдёІеҶҷжҲҗвҖңеҹҹ/дәӢд»¶еҗҚз§°вҖқпјҢдҫӢеҰӮCOUNT/INCREMENTпјҢз”ЁдәҺеҢәеҲҶдёҚеҗҢзҡ„reducerгҖӮ

еҶҚдҝ®ж”№дёҖдёӢд»Јз Ғ

зҺ°еңЁе°ұжӯЈеёёдәҶ

Redux.applyMiddleware

жҲ‘д»¬еҸҜд»ҘеңЁReduxдёӯдҪҝз”Ёеҗ„з§ҚжҸ’д»¶пјҢд»ҘеўһејәжҲ‘д»¬зҡ„еҠҹиғҪпјҢжҜ”еҰӮжҲ‘еңЁжҜҸж¬Ўдҝ®ж”№stateзҡ„ж—¶еҖҷпјҢдёҚз”ЁиҮӘе·ұжүӢеҠЁжү“еҚ°пјҢиҖҢжҳҜиҮӘеҠЁжү“еҚ°пјҢжӯӨж—¶жҲ‘д»¬еҸҜд»ҘдҪҝз”Ёredux-loggerжҸ’д»¶

Redux.bindActionCreators

дёҖиҲ¬жғ…еҶөдёӢдҪ еҸҜд»ҘзӣҙжҺҘеңЁ Store е®һдҫӢдёҠи°ғз”Ё dispatchгҖӮеҰӮжһңдҪ еңЁ React дёӯдҪҝз”Ё ReduxпјҢreact-redux дјҡжҸҗдҫӣ dispatch еҮҪж•°и®©дҪ зӣҙжҺҘи°ғз”Ёе®ғ гҖӮ

жҠҠдёҖдёӘ value дёәдёҚеҗҢ action creator зҡ„еҜ№иұЎпјҢиҪ¬жҲҗжӢҘжңүеҗҢеҗҚ key зҡ„еҜ№иұЎгҖӮеҗҢж—¶дҪҝз”Ё dispatch еҜ№жҜҸдёӘ action creator иҝӣиЎҢеҢ…иЈ…пјҢд»ҘдҫҝеҸҜд»ҘзӣҙжҺҘи°ғз”Ёе®ғд»¬гҖӮ

жғҹдёҖдјҡдҪҝз”ЁеҲ° bindActionCreators зҡ„еңәжҷҜжҳҜеҪ“дҪ йңҖиҰҒжҠҠ action creator еҫҖдёӢдј еҲ°дёҖдёӘз»„д»¶дёҠпјҢеҚҙдёҚжғіи®©иҝҷдёӘз»„д»¶и§үеҜҹеҲ° Redux зҡ„еӯҳеңЁпјҢиҖҢдё”дёҚеёҢжңӣжҠҠ dispatch жҲ– Redux store дј з»ҷе®ғгҖӮ

дёәж–№дҫҝиө·и§ҒпјҢдҪ д№ҹеҸҜд»Ҙдј е…ҘдёҖдёӘеҮҪж•°дҪңдёәз¬¬дёҖдёӘеҸӮж•°пјҢе®ғдјҡиҝ”еӣһдёҖдёӘеҮҪж•°гҖӮ

дёҠйқўзҡ„и§ЈйҮҠзҗҶи§Јиө·жқҘеҸҜиғҪжңүдәӣйҡҫд»ҘзҗҶи§ЈпјҢдёӢйқўжҲ‘д»¬з”ЁдҫӢеӯҗиҜҙжҳҺпјҡ

д»Җд№ҲжҳҜaction creatorпјҢе°ұжҳҜиҝ”еӣһactionзҡ„еҮҪж•°пјҢдҫӢеҰӮпјҡ

е®ҡд№үдәҶеҰӮдёҠaction creatorпјҢз„¶иҖҢжҲ‘д»¬жҙҫеҸ‘actionзҡ„ж—¶еҖҷд»Қз„¶йңҖиҰҒдҪҝз”Ёstore.dispatchиҝӣиЎҢжҙҫеҸ‘пјҢдҫӢеҰӮstore.dispatch(incrementActionCreator())пјҢиҝҷж ·еҶҷжңӘе…ҚжңүдәӣзҙҜиөҳдәҶпјҢжӯӨж—¶жҲ‘д»¬е°ұеҸҜд»ҘдҪҝз”ЁbindActionCreatorsж–№жі•пјҢе°Ҷaction creatorе’Ңstore.diapatchдј иҝӣеҺ»пјҢиҝ”еӣһдёҖдёӘеҮҪж•°пјҢеҸҜд»ҘзӣҙжҺҘжҙҫеҸ‘ actionпјҢдҫӢеҰӮ:

еҗҢж—¶пјҢRedux.bindActionCreatorsзҡ„з¬¬дёҖдёӘеҸӮж•°иҝҳеҸҜд»ҘжҺҘеҸ—дёҖдёӘз”ұaction creatorз»„жҲҗзҡ„еҜ№иұЎпјҢеҜ№иұЎзҡ„keyйҡҸдҫҝжҢҮе®ҡпјҢжӯӨж—¶дјҡиҝ”еӣһе’ҢkeyзӣёеҗҢзҡ„дёҖз»„еҜ№иұЎпјҢеҜ№иұЎзҡ„еҖје°ұжҳҜеҸҜд»ҘзӣҙжҺҘжҙҫеҸ‘actionзҡ„еҮҪж•°пјҢдҫӢеҰӮ:
