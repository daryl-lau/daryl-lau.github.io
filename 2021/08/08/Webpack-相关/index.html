дёүз§Қhashзӯ–з•Ҙ

  • hash

    hash е’ҢжҜҸж¬Ў buildжңүе…іпјҢжІЎжңүд»»дҪ•ж”№еҸҳзҡ„жғ…еҶөдёӢпјҢжҜҸж¬Ўзј–иҜ‘еҮәжқҘзҡ„ hashйғҪжҳҜдёҖж ·зҡ„пјҢдҪҶеҪ“дҪ ж”№еҸҳдәҶд»»дҪ•дёҖзӮ№дёңиҘҝпјҢе®ғзҡ„hashе°ұдјҡеҸ‘з”ҹж”№еҸҳгҖӮ

    з®ҖеҚ•зҗҶи§ЈпјҢдҪ ж”№дәҶд»»дҪ•дёңиҘҝпјҢhash е°ұдјҡе’ҢдёҠж¬ЎдёҚдёҖж ·дәҶгҖӮ

  • chunkhash

    chunkhashж №жҚ®дёҚеҗҢзҡ„е…ҘеҸЈж–Үд»¶(Entry)иҝӣиЎҢдҫқиө–ж–Үд»¶и§ЈжһҗгҖҒжһ„е»әеҜ№еә”зҡ„д»Јз Ғеқ—пјҲchunkпјүпјҢз”ҹжҲҗеҜ№еә”зҡ„е“ҲеёҢеҖјпјҢжҹҗж–Үд»¶еҸҳеҢ–ж—¶еҸӘжңүиҜҘж–Үд»¶еҜ№еә”д»Јз Ғеқ—пјҲchunkпјүзҡ„hashдјҡеҸҳеҢ–

  • contenthash

    е®ғзҡ„еҮәзҺ°дё»иҰҒжҳҜдёәдәҶи§ЈеҶіпјҢи®©cssж–Үд»¶дёҚеҸ—jsж–Үд»¶зҡ„еҪұе“ҚгҖӮжҜ”еҰӮfoo.cssиў«foo.jsеј•з”ЁдәҶпјҢжүҖд»Ҙе®ғд»¬е…ұз”ЁзӣёеҗҢзҡ„chunkhashеҖјгҖӮдҪҶиҝҷж ·еӯҗжҳҜжңүй—®йўҳзҡ„пјҢеҰӮжһңfoo.jsдҝ®ж”№дәҶд»Јз ҒпјҢcssж–Үд»¶е°ұз®—еҶ…е®№жІЎжңүд»»дҪ•ж”№еҸҳпјҢз”ұдәҺжҳҜиҜҘжЁЎеқ—зҡ„ hash еҸ‘з”ҹдәҶж”№еҸҳпјҢе…¶cssж–Үд»¶зҡ„hashд№ҹдјҡйҡҸд№Ӣж”№еҸҳгҖӮ

    иҝҷдёӘж—¶еҖҷжҲ‘д»¬е°ұеҸҜд»ҘдҪҝз”ЁcontenthashдәҶпјҢдҝқиҜҒеҚідҪҝcssж–Үд»¶жүҖеӨ„зҡ„жЁЎеқ—йҮҢжңүд»»дҪ•еҶ…е®№зҡ„ж”№еҸҳпјҢеҸӘиҰҒ css ж–Үд»¶еҶ…е®№дёҚеҸҳпјҢйӮЈд№Ҳе®ғзҡ„hashе°ұдёҚдјҡеҸ‘з”ҹеҸҳеҢ–гҖӮ

    contenthash дҪ еҸҜд»Ҙз®ҖеҚ•зҗҶи§ЈдёәжҳҜ moduleId + content жүҖз”ҹжҲҗзҡ„ hashгҖӮ

дјҳеҢ–зӯ–з•Ҙ

дјҳеҢ–loaderжҹҘжүҫиҢғеӣҙ

еңЁloaderдёӯз»“еҗҲtest include excludeдёүдёӘй…ҚзҪ®йЎ№жқҘзј©вј©loaderзҡ„еӨ„зҗҶиҢғеӣҙпјҢжҺЁиҚҗinclude

exclude дјҳе…Ҳзә§иҰҒдјҳдәҺ include е’Ң test пјҢжүҖд»ҘеҪ“дёүиҖ…й…ҚзҪ®жңүеҶІзӘҒж—¶пјҢ exclude дјҡдјҳе…ҲдәҺе…¶д»–дёӨдёӘй…ҚзҪ®гҖӮ

дјҳеҢ–resolve.modulesй…ҚзҪ®

resolve.modulesвҪӨдәҺй…ҚзҪ®webpackеҺ»е“ӘдәӣвҪ¬еҪ•дёӢеҜ»жүҫз¬¬дёүвҪ…жЁЎеқ—пјҢй»ҳи®ӨжҳҜ[вҖҳnode_modulesвҖҷ]

еҜ»жүҫз¬¬дёүвҪ…жЁЎеқ—пјҢй»ҳи®ӨжҳҜеңЁеҪ“еүҚйЎ№вҪ¬вҪ¬еҪ•дёӢзҡ„node_modulesвҫҘвҫҜеҺ»жүҫпјҢеҰӮжһңжІЎжңүжүҫеҲ°пјҢе°ұдјҡеҺ»дёҠвјҖзә§вҪ¬еҪ•../node_modulesжүҫпјҢеҶҚжІЎжңүдјҡеҺ»../../node_modulesдёӯжүҫпјҢд»ҘжӯӨзұ»жҺЁпјҢе’ҢNode.jsзҡ„жЁЎеқ—еҜ»жүҫжңәеҲ¶еҫҲзұ»дјјгҖӮ

еҰӮжһңжҲ‘д»¬зҡ„з¬¬дёүвҪ…жЁЎеқ—йғҪе®үиЈ…еңЁдәҶйЎ№вҪ¬ж №вҪ¬еҪ•дёӢпјҢе°ұеҸҜд»ҘзӣҙжҺҘжҢҮжҳҺиҝҷдёӘи·Ҝеҫ„гҖӮ

дјҳеҢ–resolve.aliasй…ҚзҪ®

resolve.aliasй…ҚзҪ®йҖҡиҝҮеҲ«еҗҚжқҘе°ҶеҺҹеҜјвјҠи·Ҝеҫ„жҳ е°„жҲҗвјҖдёӘж–°зҡ„еҜјвјҠи·Ҝеҫ„пјҢжӢҝreactдёәдҫӢпјҢжҲ‘д»¬еј•вјҠзҡ„reactеә“пјҢвјҖиҲ¬еӯҳеңЁдёӨеҘ—д»Јз Ғ

  • cjs

    йҮҮвҪӨcommonJSи§„иҢғзҡ„жЁЎеқ—еҢ–д»Јз Ғ

  • umd

    е·Із»Ҹжү“еҢ…еҘҪзҡ„е®Ңж•ҙд»Јз ҒпјҢжІЎжңүйҮҮвҪӨжЁЎеқ—еҢ–пјҢеҸҜд»ҘзӣҙжҺҘжү§вҫҸ

й»ҳи®Өжғ…еҶөдёӢпјҢwebpackдјҡд»ҺвјҠвјқвҪӮд»¶./node_modules/bin/react/indexејҖе§ӢйҖ’еҪ’и§Јжһҗе’ҢеӨ„зҗҶдҫқиө–зҡ„вҪӮд»¶гҖӮжҲ‘д»¬еҸҜд»ҘзӣҙжҺҘжҢҮе®ҡвҪӮд»¶пјҢйҒҝе…ҚиҝҷеӨ„зҡ„иҖ—ж—¶гҖӮ

дјҳеҢ–resolve.extensionsй…ҚзҪ®

resolve.extensionsеңЁеҜјвјҠиҜӯеҸҘжІЎеёҰвҪӮд»¶еҗҺзјҖж—¶пјҢwebpackдјҡвҫғеҠЁеёҰдёҠеҗҺзјҖеҗҺпјҢеҺ»е°қиҜ•жҹҘжүҫвҪӮд»¶жҳҜеҗҰеӯҳеңЁ

й»ҳи®ӨеҖјпјҡ

  • еҗҺзјҖе°қиҜ•еҲ—иЎЁе°ҪйҮҸзҡ„вј©
  • еҜјвјҠиҜӯеҸҘе°ҪйҮҸзҡ„еёҰдёҠеҗҺзјҖ

еҲ©вҪӨеӨҡзәҝзЁӢжҸҗеҚҮжһ„е»әйҖҹеәҰ

з”ұдәҺиҝҗвҫҸеңЁ Node.js д№ӢдёҠзҡ„ Webpack жҳҜеҚ•зәҝзЁӢжЁЎеһӢзҡ„пјҢжүҖд»Ҙ Webpack йңҖиҰҒеӨ„зҗҶзҡ„дәӢжғ…йңҖиҰҒвјҖд»¶вјҖд»¶зҡ„еҒҡпјҢдёҚиғҪеӨҡд»¶дәӢвјҖиө·еҒҡгҖӮжҲ‘д»¬йңҖиҰҒ Webpack иғҪеҗҢвјҖж—¶й—ҙеӨ„зҗҶеӨҡдёӘд»»еҠЎпјҢеҸ‘жҢҘеӨҡж ё CPU з”өи„‘зҡ„еЁҒвј’гҖӮ

thread-loaderжҳҜй’ҲеҜ№ loader иҝӣвҫҸдјҳеҢ–зҡ„пјҢе®ғдјҡе°Ҷ loader ж”ҫзҪ®еңЁвјҖдёӘ worker жұ вҫҘвҫҜиҝҗвҫҸпјҢд»ҘиҫҫеҲ°еӨҡзәҝзЁӢжһ„е»әгҖӮthread-loader еңЁдҪҝвҪӨзҡ„ж—¶еҖҷпјҢйңҖиҰҒе°Ҷе…¶ж”ҫзҪ®еңЁе…¶д»– loader д№ӢеүҚпјҢеҰӮдёӢвҫҜе®һдҫӢ:

зј“еӯҳcacheзӣёе…і

Webpack дёӯжү“еҢ…зҡ„ж ёвјјжҳҜ JavaScript вҪӮд»¶зҡ„жү“еҢ…пјҢJavaScript дҪҝвҪӨзҡ„жҳҜ babel-loaderпјҢе…¶е®һжү“еҢ…ж—¶й—ҙв»“еҫҲеӨҡж—¶еҖҷжҳҜbabel-loader жү§вҫҸж…ўеҜјиҮҙзҡ„гҖӮиҝҷж—¶еҖҷжҲ‘д»¬дёҚд»…иҰҒдҪҝвҪӨ exclude е’Ң include жқҘе°ҪеҸҜиғҪеҮҶзЎ®зҡ„жҢҮе®ҡиҰҒиҪ¬жҚўеҶ…е®№зҡ„иҢғз•ҙпјҢиҝҳйңҖиҰҒе…іжіЁ babel-loader еңЁжү§вҫҸзҡ„ж—¶еҖҷпјҢеҸҜиғҪдјҡдә§вҪЈвјҖдәӣиҝҗвҫҸжңҹй—ҙйҮҚеӨҚзҡ„е…¬е…ұвҪӮд»¶пјҢйҖ жҲҗд»Јз ҒдҪ“з§ҜвјӨеҶ—дҪҷпјҢеҗҢж—¶д№ҹдјҡеҮҸж…ўзј–иҜ‘зҡ„йҖҹеәҰгҖӮ

babel-loaderжҸҗдҫӣдәҶ cacheDirectory й…ҚзҪ®з»ҷ Babel зј–иҜ‘ж—¶з»ҷе®ҡзҡ„вҪ¬еҪ•пјҢе№¶дё”е°ҶвҪӨдәҺзј“еӯҳеҠ иҪҪеҷЁзҡ„з»“жһңпјҢдҪҶжҳҜиҝҷдёӘи®ҫзҪ®й»ҳи®ӨжҳҜ false е…ій—ӯзҡ„зҠ¶жҖҒпјҢжҲ‘д»¬йңҖиҰҒи®ҫзҪ®дёә true пјҢиҝҷж · babel-loader е°ҶдҪҝвҪӨй»ҳи®Өзҡ„зј“еӯҳвҪ¬еҪ• гҖӮnode_modules/.cache/babel-loader пјҢеҰӮжһңеңЁд»»дҪ•ж №вҪ¬еҪ•дёӢйғҪжІЎжңүжүҫеҲ° node_modules вҪ¬еҪ•пјҢе°ҶдјҡйҷҚзә§еӣһйҖҖеҲ°ж“ҚдҪңзі»з»ҹй»ҳи®Өзҡ„дёҙж—¶вҪӮд»¶вҪ¬еҪ•гҖӮ

еҺӢзј©йҖҹеәҰдјҳеҢ–

зӣёеҜ№дәҺжһ„е»әиҝҮзЁӢвҪҪвҫ”пјҢеҺӢзј©зӣёеҜ№жҲ‘д»¬жқҘиҜҙеҸӘжңүвҪЈдә§зҺҜеўғжү“еҢ…жүҚдјҡеҒҡпјҢвҪҪдё”еҺӢзј©жҲ‘д»¬йҷӨдәҶж·»еҠ  cache е’ҢеӨҡзәҝзЁӢвҪҖжҢҒд№ӢеӨ–пјҢеҸҜд»ҘдјҳеҢ–зҡ„з©әй—ҙиҫғвј©гҖӮжҲ‘д»¬еңЁдҪҝвҪӨ terser-webpack-plugin зҡ„ж—¶еҖҷеҸҜд»ҘйҖҡиҝҮдёӢвҫҜзҡ„й…ҚзҪ®ејҖеҗҜеӨҡзәҝзЁӢе’Ңзј“еӯҳпјҡ

дҪҝвҪӨexternalsдјҳеҢ–cdnйқҷжҖҒиө„жәҗ

е…¬еҸёжңүcdnпјҢйқҷжҖҒиө„жәҗжңүйғЁзҪІеҲ°cdnжңүй“ҫжҺҘдәҶпјҢжҲ‘д»¬дҪҝвҪӨcdnпјҢжҲ‘д»¬зҡ„bundleвҪӮд»¶вҫҘпјҢе°ұдёҚвҪӨжү“еҢ…иҝӣеҺ»иҝҷдёӘдҫқиө–дәҶпјҢдҪ“з§Ҝдјҡвј©еҫҲеӨҡпјҢжҲ‘д»¬еҸҜд»Ҙе°ҶвјҖдәӣJSвҪӮд»¶еӯҳеӮЁеңЁ CDN дёҠ(еҮҸе°‘ Webpack жү“еҢ…еҮәжқҘзҡ„ js дҪ“з§Ҝ)пјҢеңЁindex.htmlдёӯйҖҡиҝҮж Үзӯҫеј•вјҠпјҢеҰӮ:

жҲ‘д»¬еёҢжңӣеңЁдҪҝвҪӨж—¶пјҢд»Қз„¶еҸҜд»ҘйҖҡиҝҮ import зҡ„вҪ…ејҸеҺ»еј•вҪӨ(еҰӮ import $ from 'jquery' )пјҢе№¶дё”еёҢжңӣwebpack дёҚдјҡеҜ№е…¶иҝӣвҫҸжү“еҢ…пјҢжӯӨж—¶е°ұеҸҜд»Ҙй…ҚзҪ® externals гҖӮ

дҪҝвҪӨйқҷжҖҒиө„жәҗи·Ҝеҫ„publicPath(CDN)

CDNйҖҡиҝҮе°Ҷиө„жәҗйғЁзҪІеҲ°дё–з•Ңеҗ„ең°пјҢдҪҝеҫ—вҪӨжҲ·еҸҜд»Ҙе°ұиҝ‘и®ҝй—®иө„жәҗпјҢеҠ еҝ«и®ҝй—®йҖҹеәҰгҖӮиҰҒжҺҘвјҠCDNпјҢйңҖиҰҒжҠҠвҪ№в»ҡзҡ„йқҷжҖҒиө„жәҗдёҠдј еҲ°CDNжңҚеҠЎдёҠпјҢеңЁи®ҝй—®иҝҷдәӣиө„жәҗж—¶пјҢдҪҝвҪӨCDNжңҚеҠЎжҸҗдҫӣзҡ„URLгҖӮ

    еңЁе®һйҷ…ејҖеҸ‘дёӯпјҢжҲ‘д»¬еҸҜд»ҘйҖҡиҝҮз§»йҷӨвјҖдәӣдёҚвҪӨзҡ„д»Јз Ғд»ҺвҪҪиҫҫеҲ°дјҳеҢ–д»Јз ҒдҪ“з§Ҝзҡ„дҪңвҪӨпјҢTree-Shaking д№ҹжҳҜдҫқиө–иҝҷдёӘжҸ’д»¶зҡ„

    еҺӢзј©жҳҜеҸ‘еёғеүҚеӨ„зҗҶжңҖиҖ—ж—¶й—ҙзҡ„вјҖдёӘжӯҘйӘӨпјҢеңЁ Webpack й…ҚзҪ®дёӯеҸҜд»ҘйҖҡиҝҮејҖеҗҜ terser-webpack-plugin зҡ„еӨҡзәҝзЁӢеҺӢзј©жқҘеҠ йҖҹжҲ‘д»¬зҡ„жһ„е»әеҺӢзј©йҖҹеәҰпјҡ

еҢәеҲҶзҺҜеўғжү“еҢ…еҸҜд»ҘеҲӣе»әеӨҡдёӘдёҚеҗҢзҺҜеўғзҡ„configж–Үд»¶пјҢжү§иЎҢжү“еҢ…е‘Ҫд»Өзҡ„ж—¶еҖҷжҢҮе®ҡеҜ№еә”зҡ„configж–Үд»¶пјҢжҲ‘д»¬еҸҜд»Ҙе®ҡд№үдёҖдёӘеҹәзЎҖж–Үд»¶webpack.common.config.jsеӯҳж”ҫе…¬з”Ёзҡ„й…ҚзҪ®пјҢз„¶еҗҺеҶҚеҲӣе»әе…¶д»–зҺҜеўғзҡ„й…ҚзҪ®ж–Үд»¶пјҢдҫӢеҰӮwebpack.prod.config.jsгҖҒwebpack.dev.config.jsгҖҒwebpack.test.config.jsзӯүпјҢеңЁйҮҢйқўиҝӣиЎҢе…¬е…ұй…ҚзҪ®зҡ„еҗҲе№¶

package.js

иҝҳеҸҜд»ҘеҖҹеҠ©cross-envеҹәдәҺзҺҜеўғеҸҳйҮҸеҢәеҲҶпјҢз”ұдәҺдёҚеҗҢе№іеҸ°д№Ӣй—ҙзҺҜеўғеҸҳйҮҸзЁҚжңүдёҚеҗҢпјҢеӣ жӯӨйңҖиҰҒеҖҹеҠ©иҝҷдёӘе·Ҙе…·иҝӣиЎҢз»ҹдёҖпјҢиҝҷдёӘе·Ҙе…·еҸҜд»ҘеңЁжҲ‘д»¬жү§иЎҢжү“еҢ…е‘Ҫд»Өзҡ„ж—¶еҖҷпјҢиҮӘеҠЁдҝ®ж”№nodeзҡ„process.envзҺҜеўғеҸҳйҮҸзҡ„й…ҚзҪ®пјҢжҜ”еҰӮ

жҲ‘д»¬жү§иЎҢnpm run devзҡ„ж—¶еҖҷпјҢprocess.env.NODE_ENV иў«и®ҫзҪ®дёәdevelopmentпјҢиҖҢжү§иЎҢnpm run buildзҡ„ж—¶еҖҷprocess.env.NODE_ENV иў«и®ҫзҪ®дёәproductionпјҢз„¶еҗҺжҲ‘д»¬еҸҜд»Ҙж №жҚ®иҝҷдёӘзҺҜеўғеҸҳйҮҸпјҢеҸӘеҶҷдёҖд»Ҫwebpack.config.jsй…ҚзҪ®пјҢе®һзҺ°дёҚеҗҢзҺҜеўғзҺҜеўғзҡ„жү“еҢ…пјӣ

webpack.config.jsиҝҳеҸҜд»ҘеҜјеҮәдёҖдёӘеҮҪж•°пјҢеҮҪж•°жҺҘж”¶дёӨдёӘе…ҘеҸӮпјҡ

  • envпјҡеҪ“еүҚиҝҗиЎҢ webpack зҡ„зҺҜеўғеҸҳйҮҸ, и®ҫзҪ®е®ғж—¶йңҖиҰҒеңЁеҗҜеҠЁ webpack ж—¶еҖҷеёҰдёҠеҸӮж•°, дҫӢеҰӮ: webpack --env.production;
  • argsпјҡд»ЈиЎЁеңЁ webpack еҗҜеҠЁж—¶еҖҷйҖҡиҝҮе‘Ҫд»ӨиЎҢдј е…Ҙзҡ„жүҖжңүеҸӮж•°; дҫӢеҰӮ: --config --env --devtool зӯү;
