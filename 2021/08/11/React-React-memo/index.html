еҸ‘иЎЁдәҺ 2021-08-11 еҲҶзұ»дәҺ React жң¬ж–Үеӯ—ж•°пјҡ 430

еҰӮжһңдҪ зҡ„з»„д»¶еңЁзӣёеҗҢ props зҡ„жғ…еҶөдёӢжёІжҹ“зӣёеҗҢзҡ„з»“жһңпјҢйӮЈд№ҲдҪ еҸҜд»ҘйҖҡиҝҮе°Ҷе…¶еҢ…иЈ…еңЁ React.memo дёӯи°ғз”ЁпјҢд»ҘжӯӨйҖҡиҝҮи®°еҝҶз»„д»¶жёІжҹ“з»“жһңзҡ„ж–№ејҸжқҘжҸҗй«ҳз»„д»¶зҡ„жҖ§иғҪиЎЁзҺ°гҖӮиҝҷж„Ҹе‘ізқҖеңЁиҝҷз§Қжғ…еҶөдёӢпјҢReact е°Ҷи·іиҝҮжёІжҹ“з»„д»¶зҡ„ж“ҚдҪңе№¶зӣҙжҺҘеӨҚз”ЁжңҖиҝ‘дёҖж¬ЎжёІжҹ“зҡ„з»“жһңгҖӮ

React.memo д»…жЈҖжҹҘ props еҸҳжӣҙ

дҫӢеҰӮжҲ‘д»¬жёІжҹ“дёҖдёӘж–җжіўйӮЈеҘ‘ж•°еҲ—зҡ„з¬¬40дҪҚзҡ„еҖјпјҢиҝҷжҳҜдёҖдёӘеҫҲиҖ—ж—¶зҡ„йҖ’еҪ’ж“ҚдҪңпјҢеңЁзҲ¶з»„д»¶дёӯдҝ®ж”№stateпјҢжҲ‘д»¬зҹҘйҒ“stateеҸҳдәҶпјҢдјҡеҜјиҮҙз»„д»¶йҮҚж–°жёІжҹ“пјҢжӯӨж—¶Fibз»„д»¶д№ҹдјҡйҮҚж–°жёІжҹ“пјҢдјҡеҺ»йҮҚж–°и®Ўз®—ж–җжіўйӮЈеҘ‘ж•°еҲ—зҡ„еҖјпјҢз”ұдәҺи®Ўз®—йқһеёёиҖ—ж—¶пјҢжӯӨж—¶йЎөйқўдјҡеӨҜдҪҸпјӣ

жҹҘзңӢз»“жһңеҸ‘зҺ°пјҢжҜҸеҪ“жҲ‘д»¬зӮ№еҮ»жҢүй’®пјҢFibз»„д»¶йғҪдјҡйҮҚж–°жёІжҹ“е№¶и®Ўз®—еҖјпјҢжҖ§иғҪеҫҲе·®;

иҖҢеҰӮжһңжҲ‘д»¬дҪҝз”ЁReact.memoй«ҳйҳ¶з»„д»¶еҢ…иЈ…дёҖдёӢFibпјҢжӯӨж—¶йЎөйқўжЈҖжөӢеҲ°е®һйҷ…дёҠFibз»„д»¶зҡ„propsеұһжҖ§n е№¶жІЎжңүеҸ‘з”ҹж”№еҸҳпјҢеҲҷи·іиҝҮжёІжҹ“з»„д»¶зҡ„ж“ҚдҪңе№¶зӣҙжҺҘеӨҚз”ЁжңҖиҝ‘дёҖж¬ЎжёІжҹ“зҡ„з»“жһң

еҸӘйңҖиҰҒиҝҷд№ҲеҢ…дёҖдёӢ

йҖҡиҝҮз»“жһңеҸҜд»ҘзңӢеҲ°пјҢдҪҝз”ЁReact.memoеҢ…иЈ…дәҶд№ӢеҗҺпјҢз»„д»¶жІЎжңүйҮҚеӨҚжёІжҹ“пјҢзҲ¶з»„д»¶зҡ„иЎҢдёәд№ҹжӯЈеёё
